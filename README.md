btcTracker
==========

Btc price tracker 


Simply Ios App : Diplay some bitcoin and altcoin price ( in USD ) from the btc-e public api 

List of currency available : 

-btc -> usd
-ltc -> usd
-nmc -> usd
-nvc -> usd
-ppc -> usd


More currency and other website ( mtgox for exemple ) are coming soon.
